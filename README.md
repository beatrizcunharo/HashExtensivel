# HashExtensivel

## Como executar

* Linux

```sh
javac -d . src/hashextensivel/*.java
jar cvmf META-INF/MANIFEST.MF HashExtensivel.jar hashextensivel
java -jar HashExtensivel.jar
```

* Windows

```sh
javac -d . src\hashextensivel\*.java
jar cvmf META-INF\MANIFEST.MF HashExtensivel.jar hashextensivel
java -jar HashExtensivel.jar
```
